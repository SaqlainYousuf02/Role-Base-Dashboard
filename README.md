# Role-Based Interactive Dashboard

## Description  Live Link role-bashboard.netlify.app
This project is an **interactive dashboard** designed for **Students, Instructors, and Administrators**.  
It displays **role-based statistics** with **real-time updates**, responsive design, and dynamic cards.  
Built for learning and demonstrating **dynamic UI, animations, and role-specific data visualization**.

---

## Features & Requirements (as per task)
- Responsive design for desktop and mobile devices.
- Role-specific cards with **distinct color gradients**:
  - Student: Blue gradient
  - Instructor: Purple-Pink gradient
  - Admin: Orange-Red gradient
- Hover effects and smooth card animations.
- Dynamic statistics for each role:
  - **Student:** Enrolled Courses, Pending Assignments, New Notifications
  - **Instructor:** Active Courses, Assignments to Grade, Student Messages
  - **Administrator:** Total Courses, System Reports, System Alerts
- Cards show icons and stats in a visually appealing way.
- Refresh button to simulate live data updates:
  - Animates the refresh icon
  - Updates numbers with smooth transition
  - Shows toast notification
- Separate **HTML**, **CSS**, and **JavaScript** files:
  - `index.html` – Main structure
  - `Style.css` – Styling & animations
  - `Script.js` – Data handling & interactivity
- Professional color scheme:
  - Background: Light gradient
  - Cards: Colorful gradients per role
  - Buttons: Hover effects
- Last updated timestamp dynamically shown on dashboard.
- Fully interactive: Clicking **View Details** shows toast message for respective role.
